# Part_to_whole_analysis
Part-to-Whole Analysis concept helps to analyze how an individual part is performing compared to the overall, allowing us to understand which category has greatest impact on the buisness.

# The goal is simple:
👉 Understand how much each category contributes to the overall total.

# For example:
What percentage of total sales comes from each product category?

Which region contributes the most revenue?

Which customer segment drives the largest share of business?


Using SQL, I calculated category-wise sales and then compared them against overall sales to determine each category's contribution percentage.

# Key SQL concepts used:

✅ Aggregate Functions (SUM)

✅ Common Table Expressions (CTEs)

✅ Subqueries

✅ Percentage Calculations

✅ Business-Oriented Data Analysis

# This type of analysis helps businesses identify:
📈 Top-performing categories

📈 Revenue drivers

📈 Areas with growth potential
